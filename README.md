# Recursive Product Calculator (Java)

This project was developed as part of strengthening my understanding of recursion and core Java programming concepts. This Java program calculates the product of five user-input integers using recursion. It demonstrates how recursive functions can be used to solve mathematical problems efficiently while maintaining clean, modular code.

## Features

- Prompts user for integer input
- Uses recursion to calculate product
- Includes input validation
- Modular structure with clear function separation

## Concepts Covered

- Recursion
- Method calls and stack behavior
- Input validation
- Problem decomposition

## What I Learned

- How recursion works and how functions call themselves
- The importance of base cases to prevent infinite loops
- Structuring code into reusable methods

## 🔧 How to Run

1. Clone the repository
2. Open in a Java IDE (Eclipse, VS code, etc.)
3. Compile:

   ```bash
   javac RecursiveProduct.java
   ```
4. Run:

   ```bash
   java RecursiveProduct.java
   ```
